# Users and Roles

## Customer
Can:
- Register / login
- Browse products
- Add to cart
- Place orders
- Track orders
- Review products

## Vendor
Can:
- Register as seller
- Create products
- Manage inventory
- View and process orders
- Track earnings

## Admin
Can:
- Approve vendors
- Manage products
- Manage categories
- View orders
- Manage commissions
